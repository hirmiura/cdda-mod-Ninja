○Ninja (忍者MOD)

  [追加 - 忍者(非公式MOD)]
  (MOD識別名：Ninja)

  忍刀、黒装束などの忍者なりきりセットを追加し、忍術を強化します。
  また、NPCとして忍者が登場するようになり、彼らの出現するマップも追加されます。
  追加されたマップには、忍者だけでなく危険な罠も仕掛けられています。

  ０．注意点
  ----------
  ・新規ワールドで始めるか、ワールドのリセットが必要です。(派閥追加のため)
  ・Bandit MODに登場する忍者の装備とステータスも影響を受けます。
  ・追加アイテム名などが日本語で書かれています。(英語などでプレイしても変わりません)
  ・日本語表示でプレイしないと一部の特殊アイテムが正常に機能しません。
  ・視覚効果の関係で、タイルセット「MS32MXModded」を有効にして頂くことを推奨。
  ・忍殺要素が申し訳程度に含まれます。


  １．忍術の変更
  --------------
  ・対応武器を追加。素手格闘武器、短刀類、忍者MOD追加武器に対応
  ・効果(バフ)を一新して４種類に変更。近接スキルの上昇に応じて新しい効果をアンロック
  ・技術(攻撃方法)を３種類に変更

  <効果>
  ・近接Lv1「忍術(林)」：自動付与。近接攻撃音を無くす（バニラの"忍術"と同じ）
  ・近接Lv2「忍術(風)」：回避で付与。速度UP
  ・近接Lv3「忍術(火)」：攻撃で付与。ダメージおよび命中ボーナス
  ・近接Lv4「忍術(山)」：防御で付与。防御ボーナス
  ・近接Lv5 風・林・火・山の四種類がすべてかかると「見切り」と「暗殺」が発動するように

  <技術>
  ・近接Lv3「奇襲」　：クリティカル攻撃(バニラと同じ。ただしスタンが無くなった)
  ・近接Lv5「見切り」：回避が成功するとカウンターで攻撃。相手は転倒。ただし威力低
  ・近接Lv5「暗殺」　：相手は死ぬ

  | 見切りは体術のイメージであるため、斬撃がダメージに乗りません。打撃も威力半減です。
  |
  | 暗殺は斬撃ダメージをインフレさせて表現しています。
  | 突然、桁違いのダメージを叩き出しますが意図した動作です。
  | ダメージが一定を超えると敵が爆発四散しますが、その時の効果音が好きなので採用。
  | ちなみにインフレ率は器用依存で、斬撃武器の方がよりダメージが伸びます。
  |
  | 対応している武器、技はスタイル変更[_]の画面で[?]を押すと確認できます。
  | また、発動中の効果はゲーム中、[@]メニューで確認出来ます。


  ２．職業の追加
  ------------
  ・忍者              ：忍術を覚えており、持ち物も少し恵まれています。
  ・忍者(コスプレ)    ：黒装束セットを装備しています。刀を最初から持っているぞ。（ニヤリ
  ・ニンジャスレイヤー：特性に効果はありません。たぶん。
  ・ヤクザ            ：装備が非常に強力かつポイント消費なし。ただしNPC受けがマイナスです。

  | 忍者は忍術を最初から使えるキャラが作れるように追加しました。
  | 他の３つはおまけです。
  | なお、忍者でなくとも、「護身術」の特性を取れば忍術を習得した状態でスタートできます。

  ４．NPCの追加と変更
  -------------------
  ・派閥に「Ninja」を追加
  ・NPCに忍者(NINJA)を追加
  ・忍者クラスの装備を変更、黒装束やヌンチャク等で武装
  ・忍者クラスのステータスを強化

  | 忍者はクラスの仕様上、基本的に素手格闘で戦います。
  | 忍者は個々が強い上に徒党を組んでいるので、戦うには相応の準備が必要でしょう。
  | また、稀に強力な火器で武装し、危険な道具を使う忍者も居るかも知れません。
  |
  | 彼らは問答無用で＠を殺しに来ます。
  |
  | なお、Bandit MODで追加される忍者とは別ものです。（流派が違うと考えてください）


  ５．マップの追加
  ----------------
  ・忍者の隠れ家（家屋の地下)
  ・忍者の隠れ里（山小屋）
  ・忍者の塔（大型オフィスビル）
  ・からくり忍者屋敷（大邸宅）
  ・謎の女忍者の住むトレーラーハウス

  | 追加されたマップには忍者と罠が配置されます。また、独自の仕掛けが用意されています。
  | 罠は＠を殺すつもりで仕掛けられているので、罠スキルがある程度ないと確実に死にます。
  | 危険ですが、その分、見返りも用意されています。
  |
  | ヤクザの組事務所は単なるフレーバーです。拠点としてどうぞ。
  |
  | なお、一部のマップを除き、あえてマップ上では既存の建物と区別が付かないようにしてあります。


  ６．タイルセット
  ----------------
  MS32MXModded(盾・植木鉢・メイド・ピザ屋ロボ・PKsの新モンスター等が追加されたMShock32バリアント)
  に、当MODで追加されたタイルが収録されています。

  MS32MXModdedは本スレ or 日本語Wikiから辿って入手してください。

  注意点にも書いたとおり、いくつかの仕掛けは、このタイルセットの利用を前提としています。
  他のタイルセットでは盛大にネタバレし、アスキーモードでは逆にノーヒントになってしまいます。
  当MODをお楽しみ頂くにためにぜひ導入をお願いします。

  更にタイルが追加されたMS32MXPModdedもオススメ！
